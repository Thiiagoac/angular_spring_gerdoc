SOBRE O SISTEMA

Funcionalidade

É um sistema voltado para o gerenciamento de documentos, em que você mesmo digitaliza o documento e após ter o documento no salvo , pode-se carrega-lo no sistema para uma melhor preservação do documento e com outras opções como você categorizar o documento em tipos para maior facilidade de gerenciamento.

Tecnologia

O sistema é totalmente desenvolvido utilizando o spring boot 2.x e o Angular 8.2.x

Vantagens

Possui uma grande vantagem pela praticidade na consulta de documentos, que por muitas vezes é muito trabalhoso de procurar e achar. Já pelo sistema com apenas alguns cliques você pode achar e armazenar qualquer arquivo.
